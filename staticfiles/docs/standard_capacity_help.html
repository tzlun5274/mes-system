標準產能設定 - 操作說明

電子製造業 MES 系統 - 產能管理功能完整指南

功能概述

標準產能設定是 MES 系統的核心功能之一，用於管理產品在各個工序的標準產能數據。這些數據將用於：

  • 生產排程規劃：計算生產時間和資源需求
  • 產能分析：評估生產線效率和瓶頸
  • 成本計算：估算生產成本和工時
  • 績效評估：比較實際產能與標準產能

主要功能

新增產能設定

建立新的產品工序產能標準，包含產品編號、工序名稱、標準產能等資訊。

產能計算器

根據設備參數、作業員等級等條件，自動計算標準產能數值。

匯出 Excel

將產能設定資料匯出為 Excel 檔案，方便離線查看和編輯。

匯入 Excel

從 Excel 檔案批次匯入產能設定資料，提升資料建立效率。

操作步驟

1. 新增產能設定

  1. 點擊「新增產能設定」按鈕
  2. 填寫產品編號（必填）
  3. 選擇工序名稱（必填）
  4. 輸入標準產能數值（必填）
  5. 選擇設備類型（可選）
  6. 選擇作業員等級（可選）
  7. 設定版本號（可選）
  8. 選擇啟用狀態
  9. 點擊「儲存」完成新增

2. 使用產能計算器

  1. 點擊「產能計算器」按鈕
  2. 輸入設備參數（如：設備速度、效率係數）
  3. 選擇作業員等級
  4. 輸入產品特性（如：複雜度、尺寸）
  5. 點擊「計算」按鈕
  6. 系統會自動計算標準產能
  7. 確認結果後點擊「套用」

3. 匯出 Excel 資料

  1. 使用篩選條件選擇要匯出的資料
  2. 點擊「匯出 Excel」按鈕
  3. 選擇儲存位置
  4. 檔案會自動下載到您的電腦

4. 匯入 Excel 資料

  1. 準備符合格式的 Excel 檔案
  2. 點擊「匯入 Excel」按鈕
  3. 選擇要匯入的檔案
  4. 確認匯入設定（覆蓋或新增）
  5. 點擊「開始匯入」
  6. 查看匯入結果報告

欄位說明

欄位名稱 說明 必填 格式
產品編號 產品的唯一識別編號 是 文字（最多 50 字元）
工序名稱 生產工序的名稱 是 文字（最多 100 字元）
標準產能 每小時的標準產能數量 是 數字（最多 2 位小數）
設備類型 使用的設備類型 否 下拉選單
作業員等級 作業員的技能等級 否 下拉選單
版本號 產能設定的版本編號 否 文字（最多 20 字元）
啟用狀態 是否啟用此產能設定 是 是/否

篩選功能

使用篩選功能可以快速找到您需要的產能設定資料：

篩選條件

  • 產品編號：輸入完整或部分產品編號
  • 工序名稱：選擇特定工序
  • 設備類型：選擇特定設備類型
  • 作業員等級：選擇特定作業員等級
  • 啟用狀態：選擇啟用或停用的設定

篩選技巧

  • 可以組合多個條件進行篩選
  • 產品編號支援模糊搜尋
  • 清空所有條件可顯示全部資料
  • 篩選結果會即時更新

注意事項

重要提醒

  • 產品編號必須與 ERP 系統中的產品編號一致
  • 標準產能數值必須大於 0
  • 同一產品同一工序只能有一個啟用的產能設定
  • 停用產能設定前請確認沒有相關的生產排程

使用技巧

  • 建議定期更新產能設定以反映實際生產狀況
  • 可以使用版本號來管理不同時期的產能設定
  • 匯入 Excel 前請先備份現有資料
  • 產能計算器可以幫助您快速建立合理的產能設定
