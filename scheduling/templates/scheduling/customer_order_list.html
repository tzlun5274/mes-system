客戶訂單管理

本頁顯示所有公司國內/國外訂單（同步自ERP），可依公司、客戶、訂單類型、交期查詢，並可匯出Excel。

同步條件說明
國內訂單同步條件
  • 訂單號碼格式：113-*
  • 訂單日期：2020年1月1日 以後
  • 訂單狀態：未結案 (BillStatus = 0 或 NULL)
  • 未交數量：> 0
  • 產品編號：PFP-* 開頭

資料來源： ordBillMain (主表) + ordBillSub (明細表) + comCustomer (客戶表)

國外訂單同步條件
  • 訂單號碼格式：542-*
  • 訂單日期：2020年1月1日 以後
  • 訂單狀態：未結案 (BillStatus = 0 或 NULL)
  • 未交數量：> 0
  • 產品編號：PFP-* 開頭

資料來源： TraBillMain (主表) + TraBillSub (明細表) + comCustomer (客戶表)

重要說明
  • 同步頻率： 每30分鐘自動同步一次，也可手動執行同步
  • 資料更新： 每次同步會清空現有資料，重新從ERP提取
  • 公司範圍： 僅同步已設定在ERP整合模組中的公司資料
  • 資料庫連線： 使用各公司專屬的MES資料庫進行查詢
  • 狀態判斷： 逾期訂單（交期 < 今天）、緊急訂單（交期 ≤ 7天內）
