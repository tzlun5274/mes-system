CSV 格式說明
必要欄位
  • Subject：假期名稱（必填）
  • Start Date：開始日期（必填）
  • End Date：結束日期（必填）
  • All Day Event：是否為全天事件（建議設為 TRUE）
日期格式
  • YYYY/MM/DD：2026/01/01
  • YYYY-MM-DD：2026-01-01
  • YYYY/M/D：2026/1/1
  • YYYY-M-D：2026-1-1
注意事項
  • 檔案必須是 UTF-8 編碼
  • 第一行必須是欄位標題
  • 日期格式必須正確
  • 假期名稱不能為空
  • 已存在的假期會被跳過
建議格式
  • 使用標準的國定假日名稱
  • Start Date 和 End Date 通常相同
  • All Day Ev 設為 TRUE
  • 可以在 Description 欄位添加說明
